## vue-slider(vue滑动组件)

基于 `vue`的简单滑动组件，

### how to use

首页安装组件

```
npm i xcxerxes-vue-slider or yarn add xcxerxes-vue-slider
```

在根目录中 `./src/main.js` 中

```
import xcxerxesVueSlider from 'xcxerxes-vue-slider'

Vue.component(xcxerxesVueSlider.name, xcxerxesVueSlider)
```
